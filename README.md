[http://www.cerb5blog.com/](http://www.cerb5blog.com/)  

What's this?
------------
Allow you to optionally add additional items to the audit log & update the tickets update field on posts other then new emails

Installation
------------
* Change directory to **/cerb5/storage/plugins/**
* `git clone git://github.com/rmiddle/cerb5blog.last_action_and_audit_log.git`
* In your helpdesk, enable the plugin from **Setup->Features & Plugins**.
